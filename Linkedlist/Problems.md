 Basic Linked List Questions
Traverse a linked list and print all elements.
(Example) Given head -> 5 -> 4 -> 3 -> 1 -> 0, output: [5, 4, 3, 1, 0].

Find the length of a linked list.
(Example) Given head -> 1 -> 2 -> 3, output: 3.

Search for an element in a linked list.
(Example) Given head -> 7 -> 2 -> 9 -> 4, search for 9, output: true.

Insert a node at the beginning of a linked list.
(Example) Insert 5 in head -> 1 -> 2, output: 5 -> 1 -> 2.

Insert a node at the end of a linked list.
(Example) Insert 6 in head -> 1 -> 2, output: 1 -> 2 -> 6.

Insert a node at a given position in a linked list.
(Example) Insert 9 at position 2 in head -> 1 -> 2 -> 3, output: 1 -> 9 -> 2 -> 3.

Delete a node from the beginning of a linked list.
(Example) Given head -> 3 -> 4 -> 5, output: 4 -> 5.

Delete a node from the end of a linked list.
(Example) Given head -> 3 -> 4 -> 5, output: 3 -> 4.

Delete a node from a given position in a linked list.
(Example) Remove node at position 2 in head -> 1 -> 2 -> 3 -> 4, output: 1 -> 3 -> 4.

Reverse a linked list.
(Example) Given head -> 1 -> 2 -> 3, output: 3 -> 2 -> 1.
